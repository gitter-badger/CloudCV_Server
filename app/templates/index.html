News

  • 1/31/2014: NVIDIA supports CloudCV with a GPU hardware donation.
  • 1/17/2014: CloudCV recieves Microsoft Azure Research Award
  • 10/15/2013: CloudCV has been awarded Amazon Web Services (AWS) in Education: Machine Learning Research Grant.


This is a project being developed at the Machine Learning and Perception Lab, led by Prof. Dhruv Batra



CloudCV uses GraphLab for implementing distributed computer vision algorithms.

Some of the functionality is derived from using OpenCV with Graphlab SDK.
