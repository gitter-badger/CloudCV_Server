This document describes the REST API and resources provided by CloudCV. CloudCV’s REST APIs provide access to resources (data entities) via URI paths.

To use a REST API, your application will make an HTTP request and parse the response. The response format is JSON. Your methods will be the standard HTTP methods like GET, PUT, POST and DELETE.

Because the REST API is based on open standards, you can use any web development language to access the API.
